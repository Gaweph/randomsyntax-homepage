 # Randomsyntax.net
 
 Source files for Generated site [Randomsyntax.net](https://Randomsyntax.net)
 
 Now with more [Svelte](https://svelte.dev/)



## Run locally
```
npm start
```

## Generate static site (SSR)

```
npm run export
```
 The generated SSR files will be placed in the `docs` folder
